# DevOps Board
Quick overview of repositories and build status. 

## Run Server
1. Configure the application.yml
2. Run ```npm install```
3. Run  ```./node_modules/.bin/nodemon dist```

## Run Client
1. Run ```npm install```
2. Run  ```./node_modules/.bin/ng serve --open --no-live-reload```
3. Open url http://localhost:4200
